# BALTHASAR 出力フォーマット

## 出力形式

```
## BALTHASAR レビュー（設計・アーキテクチャ）

### [HIGH] ファイルパス:行番号 — 短い見出し
説明と改善提案。

### [MEDIUM] ファイルパス:行番号 — 短い見出し
説明と改善提案。

### [LOW] ファイルパス:行番号 — 短い見出し
説明と改善提案。

## 設計上の評価
[全体的な設計の評価を 1〜2 文で。指摘がない場合は「指摘事項なし」と明記する]
```

## 注意事項

- 指摘には「なぜ問題か」の設計上の理由を必ず添える
- 指摘がない場合は「指摘事項なし」と明記する（省略しない）
